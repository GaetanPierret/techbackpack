La ligne de commande

C'est une interface en mode texte uniquement. Il peut servir a lancer des programmes, a manipuler des fichiers, a mettre à jour le systéme .... Un programmeur web intéragit avec les serveurs depuis un terminal en ligne de commande. Il va en particulier deployer le site sur le serveur en utilisant des scripts et des outils depuis la ligne de commande.

Git

  • Permet de travailler à plusieurs sur le même fichier
  • Suivre l’évolution d’un code source

Git

  • git pull permet de récupérer la derniere version a date sur le dépot
  • git push permet d'envoyer des fichiers sur le dépot
  • git commit​ intègre une nouvelle version des fichiers avec un message

La balise Head

Toutes les informations nécessaires qui ne sont pas directement affichéee

  • Le titre du navigateur
  • L'auteur, la description
  • Les liens vers les fichiers externes CSS / JS
  • Les codes javascript

Les sections d'une page HTML

  • Le header : <header>
  • La navigation bar : <nav>
  • Le contenu principal : <main>, <article>, <section>, et <div>.
  • La sidebar : <aside>
  • Le footer : <footer>
